# AutoRepo 功能总结

> **版本**: v2.1  
> **更新日期**: 2026-02-03

---

## 核心功能

AutoRepo 是一款 Git 风格的车辆维护追踪系统。

| Git 概念 | AutoRepo 对应 |
|---------|--------------|
| Repository | 你的车辆 |
| Commit | 维护/改装记录 |
| Issue | 计划中的保养任务 |
| HEAD | 当前车况（里程、状态） |

---

## 功能列表

### 用户认证 🔐

- 微信一键登录（自动调用）
- JWT 令牌认证（7 天有效期）
- 多租户数据隔离
- 遗留数据自动迁移

### 车辆管理 🚗

- 多车辆支持（无限制）
- 车辆信息：名称、车架号、颜色、里程
- 保险/年检到期提醒
- 左滑删除

### 维护记录 📝

- Git 风格时间线展示
- 3 种记录类型：保养 / 维修 / 改装
- 费用追踪：零件费 + 工时费
- 图片上传（每条最多 9 张）
- 快速模板填充
- 编辑/删除功能

### 待办任务 (Issues) ✅

- 创建计划保养任务
- 设置截止日期或目标里程
- 优先级管理（高/中/低）
- 左滑删除、重新打开
- 完成任务自动关联记录

### 数据洞察 📊

- 总费用统计
- 每公里成本计算
- 费用构成分析（按类型）
- 月度趋势图表

### 导出功能 📄

- PDF 报告导出（Base64 编码）
- 包含车辆信息和完整维护历史
- 中文字体支持

### 用户体验 🎨

- 深色模式（跟随系统）
- 玻璃态设计风格
- 自定义导航栏
- 骨架屏加载
- 下拉刷新

## 技术栈

| 层级 | 技术 |
|------|------|
| 前端 | TypeScript, SCSS, 微信小程序 |
| 后端 | Python 3.9+, FastAPI, Pydantic |
| 数据库 | MongoDB / MockDB |
| 认证 | JWT + 微信登录 |
| PDF | ReportLab |
| 部署 | Docker, 微信云托管 |
